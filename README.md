# 🔥 FIRE Path Planner

A smart, India-focused financial planning web app that helps users achieve **Financial Independence & Retire Early (FIRE)** with a clear month-by-month roadmap.

---

## 🚀 Features

* 📊 **FIRE Goal Calculation**

  * Calculates required corpus based on user inputs
* 💰 **Monthly SIP Recommendation**

  * Suggests how much to invest monthly
* 📈 **Projected Corpus**

  * Estimates future wealth using expected returns (XIRR)
* 🧾 **Tax Saving Insights**

  * Provides savings under 80C & 80CCD(1B)
* 🧠 **Risk Profile Selection**

  * Conservative / Moderate / Aggressive planning
* 🎯 **Life Goal Integration**

  * Customize plans (Home, FIRE, etc.)
* 📅 **Month-by-Month Roadmap**

  * Actionable financial steps

## 📊 Input Parameters

| Field            | Description         |
| ---------------- | ------------------- |
| Current Age      | User's present age  |
| Target FIRE Age  | Retirement goal age |
| Monthly Income   | Income in ₹         |
| Monthly Expenses | Expenses in ₹       |
| Existing Corpus  | Current savings     |
| SIP              | Monthly investments |
| Risk Profile     | Investment style    |
| Salary Growth    | Annual growth %     |

---

## 🧠 How It Works

1. Collects financial inputs from user
2. Calculates:

   * Required corpus
   * Monthly SIP
   * Investment growth
3. Generates:

   * FIRE number
   * Monthly roadmap
4. Displays results in interactive dashboard
